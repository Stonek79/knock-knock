# Roadmap проекта Knock-Knock

Актуальный статус разработки на **Февраль 2026**.

## 🟢 Выполнено (Done)

### 1. Инфраструктура и Настройка
- [x] Инициализация репозитория и монорепозитория (`/app`, `/infra`, `/docs`).
- [x] Настройка линтера Biome и Husky (pre-commit hooks).
- [x] Настройка Vite + TypeScript + React 19.
- [x] Подключение Radix UI и тем.

### 2. Базовой Функционал
- [x] **Роутинг**: TanStack Router (File-based routing).
- [x] **i18n**: Локализация (RU/EN), автоопределение языка.
- [x] **State Management**: Zustand (+ devtools).
- [x] **Backend**: Подключение клиента Supabase.
- [x] **Realtime**: Подписки на изменения БД (сообщения, чаты).
- [x] **Presence**: Индикаторы "В сети" и "Печатает...".

### 3. UI/UX и Брендинг
- [x] **Логотип**: Дизайн логотипа "Knock-Knock" (Дверь/Кулак), адаптация под светлую тему.
- [x] **Фавикон**: Векторная (SVG) иконка для браузера.
- [x] **Лендинг**: Стартовая страница (`/`) с приветствием.
- [x] **Компоненты**: `AppLogo`, `EmailInput`, `PasswordInput`, `Alert`, `Button`.
- [x] **Toast Notifications**: Система уведомлений (`useToast`).

### 4. Авторизация (Auth) и Безопасность
- [x] Страница входа (`/login`).
- [x] Вход по Magic Link (Email).
- [x] Вход по Паролю.
- [x] Валидация форм (TanStack Form + Zod).
- [x] **Ghost Mode**: PIN-код, ложный вход (Decoy PIN).

### 5. Чаты (Messaging)
- [x] `ChatList`: Список чатов с поиском и фильтрацией.
- [x] `ChatRoom`: Экран переписки.
- [x] `Favorites`: Сохраненные сообщения (Self-Chat).
- [x] Отправка/получение текстовых сообщений.
- [x] Индикатор набора текста (`TypingIndicator`).
- [x] Статус онлайн (`OnlineStatus`) в личных сообщениях.

### 6. End-to-End Шифрование (E2E)
- [x] Генерация ключей (Web Crypto API) - `useKeystore`.
- [x] Автосинхронизация публичных ключей с профилем (`useKeySync`).
- [x] Обмен ключами через сервер (ECDH P-256).
- [x] Прозрачное шифрование/расшифровка сообщений (AES-GCM 256).
- [x] Wrap/Unwrap Room Keys для участников чата.
- [x] Восстановление ключей из бэкапа (Password-based).

### 7. Настройки
- [x] Страница настроек (`/settings`).
- [x] Переключатель темы (Dark/Light).
- [x] Настройки уведомлений (UI).
- [x] Настройки приватности (UI).
- [x] Настройки безопасности (UI, экспорт/импорт ключей).

### 8. Архитектура и Рефакторинг (FSD)
- [x] Строгая изоляция фичей (устранение кросс-импортов).
- [x] Выделение бизнес-логики из слоя Layouts.
- [x] Декомпозиция огромных файлов сервисов (`room.ts`).
- [x] Структурирование крупных фичей на саб-домены (`chat/room`, `chat/message`).
- [x] Удаление хардкода в CSS (переход на токены Radix).
