## 2. Улучшение Message Input (Авто-высота)
### Проблема
Текущий расчет высоты через JS (`scrollHeight`) хрупок и требует `setTimeout`.

### Решение: CSS Grid Trick
Перейти на декларативный подход расширения через CSS Grid.
1. Создать компонент-обертку в `MessageInput`.
2. Использовать невидимый `span` (реплику текста) для задания высоты контейнера.
3. `TextArea` будет занимать всю площадь контейнера.
